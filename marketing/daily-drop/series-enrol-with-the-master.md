# Daily Drop — series: "Enrol with the Master"

A themed run inside the locked Daily Drop format (see `FORMAT-LOCK.md`). Each post is a wordless
process clip + the 6-part caption, pointing at exactly ONE atlas page that carries the new
**"Enrol with the master"** badge. The badge IS the campaign: the one thing ET can say that a
travel account, MasterClass, or a listicle cannot — *a named master who will actually teach you.*

Voice lock: connect/introduce, never sell. No prices. No emoji. Banned: transformation,
life-changing, vacation, luxury, easy. Footage: licensed/reposted from each school with credit
(= partner recruitment, see `OUTREACH.md`).

Distribution per post: same clip → TikTok + Instagram Reels + YouTube Shorts, same day,
@educatedtraveler.app. Watch saves, profile taps, atlas clicks, Circle signups — not follower count.
