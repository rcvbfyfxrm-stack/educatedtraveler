### Module 9: PSYCHOLOGY & PERSUASION

**Invoke**: `NEXUS: Psychology + [Challenge]`

#### System 1 vs System 2 (Kahneman)

| System 1 (Fast) | System 2 (Slow) |
|-----------------|-----------------|
| Automatic | Deliberate |
| Intuitive | Logical |
| Emotional | Analytical |
| Makes most decisions | Rationalizes afterward |

**Implication**: Design for System 1. Logic comes later.

#### Loss Aversion

Losing $100 hurts 2x more than gaining $100 feels good.

**Application**:
- "Don't miss the March cohort" > "Join the March cohort"
- "Last 3 spots" creates loss frame
- "Your reserved spot expires in 48 hours"

#### Anchoring Effect

First number presented becomes reference point.

**Application**:
- Show $16,500 value BEFORE $4,500 price
- Compare to $30k culinary school
- "From $214/day" anchors low

#### The Hook Model (Eyal)

```
TRIGGER → ACTION → REWARD → INVESTMENT
   │          │         │          │
External   Minimum   Variable    Profile,
(email)    effort    (stories,   history,
Internal   (click,   community)  preferences
(boredom)  scroll)
```

**Our Loop**:
- Trigger: Email about new cohort
- Action: Visit landing page
- Reward: Inspiring transformation stories
- Investment: Application, deposit

#### Cognitive Biases to Leverage

| Bias | Application |
|------|-------------|
| Anchoring | High value before price |
| Decoy | Three tiers — middle wins |
| IKEA Effect | Co-create experience → value it more |
| Endowment | "Your spot is reserved" |
| Bandwagon | "Join 200+ alumni" |
| Authority | Logos, credentials, instructor bios |

1. Check funnel — where's the drop-off?

THIS WEEK:
4. A/B test headline
5. Add testimonial near CTA
6. Create urgency mechanism
```

```
IMMEDIATE:
1. Respond personally within 1 hour
2. Take conversation private

THIS WEEK:
4. Resolve root issue
5. Follow up with resolution
